## 这份指南怎么用

- 新住户：先看完整指南里的“入住前 30 分钟必看”和“日常生活速查”。
- 老住户：发现不准的信息，可以按模板补充。
- 群管理员/维护者：优先更新“待确认清单”，把不确定信息和已确认信息分开。

## 信息可信度标记

| 标记 | 含义 |
|---|---|
| 官方信息 | 来自政府网站、城投相关公开信息或现场正式公示 |
| 住户实测 | 来自住户实际体验、点餐截图、现场观察 |
| 待核实 | 目前有线索，但缺少稳定来源或近期确认 |

## 隐私边界

本仓库只保存公开、可分享的社区生活信息，不保存也不发布个人合同、身份证、手机号、具体房号、租金、备案编号、税务清单等私人资料。

如果你要补充信息，请避免上传含个人信息的截图。可以把图片中的姓名、电话、房号、二维码、订单号等打码后再提交。

## 维护原则

1. 先实用，后精致。
2. 公开信息和个人隐私硬隔离。
3. 能引用来源的写来源，不能确认的标“待核实”。
4. 不把个别体验包装成普遍结论。
5. 让新住户能在 5 分钟内找到下一步动作。
