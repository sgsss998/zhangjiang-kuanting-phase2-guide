## 一、这个文档是做什么的

欢迎加入张江城投宽庭二期。

为了让新入住的邻居少走弯路，也方便大家互相照应，我们整理了这份生活指南。内容会持续更新，主要包括吃饭、停车、健身、干洗、垃圾投放、快递外卖、周边便利设施等日常信息。

如果你发现信息不准确，或者有新的好用地点、避坑经验、生活建议，欢迎在群里补充。我们会定期整理进文档。

## 三、吃饭与周边餐饮

### 1. 康婕社区食堂

适合：日常工作餐、懒得做饭、想吃便宜正餐。

目前已知信息：

- 价格不贵，适合日常吃。
- 可以办卡，300 元办卡后约 85 折。
- 适合长期住附近的人作为稳定饭点。

待补充：

- 具体营业时间
- 具体位置截图
- 推荐菜
- 是否支持外带/打包

### 2. 秀沿路夜市一条街

适合：晚上吃饭、随便逛逛、想吃小吃或快餐。

距离：步行约 500 米左右，比较近。

目前已知有：

- 蜜雪冰城
- 麦当劳
- 肯德基
- 涮肉
- 羊肉串
- 313 羊庄
- 其他小吃和餐饮

这个区域比较适合晚上觅食，选择多，生活气息也比较足。

### 3. 一期附近餐饮

一期附近目前已知有：

- 汉堡王
- Tim Hortons
- 其他待补充

适合白天咖啡、简餐、临时办公或约人碰面。

## 四、停车与充电

### 1. 停车

目前已知：

- 停车费约 300 元/月。

建议需要长期停车的邻居，优先向物业确认：

- 月租办理方式
- 是否需要车位登记
- 是否有固定车位
- 临停收费标准
- 地面/地下停车规则

### 2. 充电桩

目前已知：

- 一期停车场路面有充电桩。
- 价格约 1.3 元/度，个人感觉不算便宜。
- 二期暂时还没有明确找到可用充电点。

待补充：

- 二期是否有地下/地面充电桩
- 充电桩品牌
- 是否需要 App
- 峰谷电价情况
- 是否容易排队

## 五、健身房

目前已知：

- 位置：一期 2 号楼。
- 价格：约 365 元/年。
- 办理方式：需要去健身房里面找教练付款。
- 权限开通：需要去一期物业处开通人脸。

适合：想低成本日常锻炼的邻居。

待补充：

- 健身房营业时间
- 器械情况
- 是否有淋浴
- 高峰期拥挤程度
- 是否限制二期住户使用
