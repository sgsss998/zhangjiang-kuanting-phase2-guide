# 信息补充模板

复制下面模板发到群里即可。

```text
信息类别：
地点/事项：
具体位置：
价格/时间：
来源：
个人评价：
是否推荐：
待核实：
是否可公开：
```

## 示例 1：餐饮

```text
信息类别：吃饭
地点/事项：某某面馆
具体位置：秀沿路附近
价格/时间：人均 25 元左右
来源：本人堂食
个人评价：出餐快，适合工作日晚饭
是否推荐：推荐
待核实：营业时间
是否可公开：可以
```

## 示例 2：快递/外卖

```text
信息类别：快递外卖
地点/事项：外卖放置点
具体位置：X 号楼一楼货架
价格/时间：全天
来源：本人日常取外卖
个人评价：晚高峰容易堆满
是否推荐：无
待核实：是否为物业指定位置
是否可公开：可以，但不要写具体房号
```

## 示例 3：隐私材料处理

如果截图里有姓名、手机号、房号、订单号、付款码、身份证号，请先打码，或者只发文字摘要。
