# 资料来源与更新规则

## 信息分级

| 等级 | 使用方式 | 示例 |
|---|---|---|
| 官方信息 | 可作为稳定背景信息引用 | 政府网站、城投公开稿、现场正式公示 |
| 住户实测 | 可写入正文，但要标注时间和来源类型 | 菜单截图、现场拍照、群友体验 |
| 待核实 | 放入待确认清单，不作为确定结论 | 听说某处有充电桩、某店将开业 |

## 隐私规则

不得进入本仓库的内容：

- 个人租赁合同
- 身份证、手机号、邮箱、微信号
- 具体房号
- 租金、面积、合同周期
- 备案编号、税务清单
- 订单号、付款码、个人二维码

允许进入本仓库的内容：

- 公开地址
- 商户菜单
- 公共配套
- 群友脱敏后的体验
- 公开官方链接

## 更新流程

1. 新信息先判断是否含隐私。
2. 含隐私的资料不进仓库，只提炼公开安全信息。
3. 可公开资料按板块归档。
4. 正文中给出信息来源和更新日期。
5. 发布前执行隐私关键词扫描。
