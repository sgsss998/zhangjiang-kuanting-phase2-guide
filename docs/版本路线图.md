# 版本路线图

> 路线图按“新住户真正需要什么”倒推，不追求一次写满，优先让每一版都更可用。

## 当前版本：3.x

目标：搭好公开指南骨架，建立可信度标记和隐私边界。

已完成：

- README 项目首页。
- 完整生活指南 3.0。
- 资料来源与更新规则。
- 待确认清单。
- 信息补充模板。
- 湘八爷商户页和菜单图片。
- 官方信源引用。

3.x 后续重点：

- 补齐外卖、快递、垃圾、物业、报修等高频刚需信息。
- 把所有“待补充”变成可执行的现场采集任务。
- 开始制作微信群 PDF 版的结构样稿。

## 4.0：入住第一周指南

目标：让新住户按时间线完成入住适应。

拟新增结构：

- 入住当天 checklist。
- 入住第一晚：吃饭、外卖、垃圾、门禁。
- 入住前三天：快递、报修、物业、生活缴费。
- 入住第一周：健身、洗衣、停车、充电、周边采购。
- 新住户常见问题 FAQ。

需要资料：

- 各楼栋外卖点。
- 快递柜/驿站点位。
- 物业报修入口。
- 垃圾投放时间和位置。
- 门禁/访客流程。

## 5.0：楼栋化和点位化

目标：从“文字指南”升级为“按楼栋查信息”。

拟新增结构：

- 1-15 号楼点位表。
- 外卖放置点表。
- 垃圾投放点表。
- 快递/驿站/临停点表。
- 停车与充电点位表。

需要资料：

- 每栋楼一楼现场照片。
- 每栋楼垃圾点位置。
- 快递柜与货架照片。
- 停车出入口和充电桩照片。

## 6.0：周边生活地图

目标：让住户按需求找到附近最实用的地方。

拟新增板块：

- 5 分钟吃饭地图。
- 10 分钟采购地图。
- 洗衣、药店、打印、维修、配钥匙。
- 通勤路线。
- 夜间路线和安全感反馈。

需要资料：

- 商户菜单和实拍。
- 营业时间。
- 人均价格。
- 步行时间。
- 是否推荐。

## 7.0：社群与活动版

目标：让指南从“生活工具”升级为“社区连接器”。

拟新增板块：

- 饭搭子。
- 健身搭子。
- 羽毛球、桌游、跑步、散步。
- 闲置转让。
- 拼车/拼单。
- 宠物友好。

需要资料：

- 群内活动反馈。
- 可公开的活动规则。
- 不扰民、不刷屏、不商业化的社群边界。

## 8.0：视觉优化版

目标：形成适合微信群传播的精美 PDF/Word。

拟交付：

- 封面。
- 目录。
- 速查表。
- 图文混排。
- 信息卡片。
- 待补充二维码或链接。

需要资料：

- 社区公共区域照片。
- 可公开的商户图片。
- 可公开的设施照片。
- 群公告最终口径。

## 10.0：成熟社区知识库

目标：形成长期可维护、可协作、可版本化的社区生活知识库。

成熟标准：

- 每个高频问题都有明确答案或待确认责任。
- 每个商户有独立卡片。
- 每个楼栋有点位表。
- 每次更新有版本记录。
- Markdown、PDF、Word 三种形态同步。
- 隐私扫描成为发布前固定动作。

## 当前派给资料提供者的任务

优先级最高：

1. 各楼栋外卖放置点。
2. 快递柜/驿站位置。
3. 物业前台、报修、管家联系方式公示。
4. 二期每栋楼垃圾点位置和投放时间。
5. 二期是否有可用充电桩。
6. 健身房营业时间、器械、是否有淋浴。

资料提供方式不限：照片、截图、语音转文字、地址、菜单、群友反馈都可以。维护者负责整理和发布。
